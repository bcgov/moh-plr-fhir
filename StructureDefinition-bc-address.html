Data Type Profile: BCAddress

Official URL: http://hl7.org/fhir/ca-bc/provider/StructureDefinition/bc-address Version: 0.1.0
Active as of 2022-10-28 Computable Name: BCAddress

Adds an end reason, communication purpose and validation status to the address.

Usage:

  • Use this DataType Constraint: BC Facility Mailing Address Extension, BCLocation, BCOrganization, BCPractitioner
